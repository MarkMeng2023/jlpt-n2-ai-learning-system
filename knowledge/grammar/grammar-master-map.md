## 易混组

- **～ものの ↔ ～にもかかわらず**：两者均表示让步；后者更强调不受条件影响。
- **～ながらも ↔ ～つつ（も）**：つつも通常接动词ます形并带持续或明知故犯语感。
- **～反面 ↔ ～一方で**：反面强调同一事物两面，一方で也可对照不同主体。
- **～どころか ↔ ～ばかりか**：どころか可推翻前项，ばかりか是在前项上递进。
- **～ことから ↔ ～ものだから**：ことから说明客观依据，ものだから常带主观解释或辩解。
- **～ものだから ↔ ～ばかりに**：ばかりに强调单一原因带来不良结果。
- **～ようだ ↔ ～らしい**：ようだ偏观察判断，らしい也可来自传闻。
- **～おそれがある ↔ ～かねない**：两者都表示风险；おそれがある更客观正式。
- **～しかない ↔ ～ざるを得ない**：均可译为只好；ざるを得ない更正式且强调被迫。
- **～に限って ↔ ～に限らず**：に限って表示唯独或偏偏，に限らず表示不限于。
- **～わけではない ↔ ～ないことはない**：前者部分否定结论，后者双重否定表示并非不可能。
- **～ものなら ↔ ～ようものなら**：可能形ものなら多为难实现愿望，意向形ようものなら多为严重后果。
- **～ために ↔ ～ように**：ために常接意志目标，ように常用于能力或非意志状态。
- **～に伴って ↔ ～につれて**：均表示伴随变化；につれて更强调连续自然变化。
- **～につれて ↔ ～に従って**：に従って还可表示遵照规则。
- **～とともに ↔ ～に伴って**：とともに还可表示共同主体或同时发生。
- **～ことになる ↔ ～ことにする**：ことになる偏外部决定，ことにする偏个人决定。
- **～ざるを得ない ↔ ～ないわけにはいかない**：前者受客观情况所迫，后者常因责任或社会常识。
- **～ほど ↔ ～くらい／ぐらい**：都可表示程度；ほど还用于比较和越……越……。
- **～に比べて ↔ ～に対して**：に比べて强调比较，に対して也可表示对象或对照。
- **～わりに ↔ ～にしては**：均表示不符合预期；にしては以身份或阶段为评价基准。
- **～どころではない ↔ ～どころか**：どころではない表示无暇做，どころか表示反转或递进。
- **～最中に ↔ ～ところに／へ／を**：最中に强调动作正进行，ところに可对应开始、进行或完成阶段。
- **～に先立って ↔ ～に際して**：に先立って明确在前，に際して表示在特殊时机。

## 其他强弱与变体关系

- **～ものの → ～ながらも**（similar）：都可表示虽然，但ながらも还可保留状态并行感。
- **～とはいえ → ～ものの**（similar）：都先承认前项再转折，とはいえ更常补充限制。
- **～くせに → ～ものの**（stronger_than）：くせに比ものの更强烈地表达责备。
- **～に反して → ～にもかかわらず**（similar）：均涉及预期与结果不一致，但接续和焦点不同。
- **～おかげで → ～せいで**（contrast）：おかげで通常导向积极结果，せいで导向消极结果。
- **～あまり → ～ばかりに**（similar）：两者都可说明不正常结果，但あまり强调程度过高。
- **～あげく → ～ばかりに**（similar）：あげく强调长期过程后的结果，ばかりに强调原因。
- **～以上（は） → ～べきだ**（similar）：以上は常引出责任或べきだ等当然结论。
- **～に違いない → ～らしい**（stronger_than）：に違いない确信度通常高于らしい。
- **～に決まっている → ～に違いない**（casual_variant）：に決まっている更主观且较口语。
- **～はずだ → ～わけがない**（contrast）：はずだ表示合理预期，わけがない强烈否定可能。
- **～まい → ～わけがない**（similar）：两者可表达否定判断，但まい还可表示否定意志。
- **～にほかならない → ～にすぎない**（contrast）：前者强调正是，后者限制为不过是。
- **～さえ → ～ばかりか**（similar）：さえ举极端例，ばかりか连接递进的两项。
- **～ばかりか → ～のみならず**（formal_variant）：のみならず是更正式的书面递进形式。
- **～さえ～ば → ～ない限り**（contrast）：さえ～ば提出最低充分条件，ない限り提出必要条件。
- **～としたら → ～とすれば**（similar）：均表示假设，とすれば更书面且偏逻辑推论。
- **～限り → ～ない限り**（contrast）：前者表示只要状态持续，后者表示除非条件实现。
- **～ないことには → ～ない限り**（similar）：都表示前项是后项成立的必要条件。
- **～べく → ～ために**（formal_variant）：べく是表示目的的正式书面形式。
- **～に向けて → ～ために**（similar）：に向けて强调朝目标推进准备。
- **～に備えて → ～に向けて**（similar）：に備えて专指为未来情况预先准备。
- **～ようになる → ～ことにする**（contrast）：ようになる是状态变化，ことにする是主动决定。
- **～ばかりだ → ～につれて**（similar）：ばかりだ强调单向持续趋势，につれて强调两项同步变化。
- **～べきだ → ～べきではない**（contrast）：分别表示应该与不应该。
- **～ことだ → ～べきだ**（weaker_than）：ことだ通常是较柔和的忠告，べきだ规范性更强。
- **～ことはない → ～べきではない**（contrast）：ことはない表示没必要，べきではない表示不应该。
- **～てはならない → ～べきではない**（stronger_than）：てはならない是正式禁止，强于建议性的べきではない。
- **～ことになっている → ～てはならない**（similar）：制度规定常与正式禁止表达共同出现。
- **～うちに → ～て以来**（contrast）：うちに限定一段期间，て以来表示从起点持续至今。
- **～たびに → ～限り**（contrast）：たびに表示每次重复，限り表示条件持续期间。
